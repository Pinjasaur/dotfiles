### Tmux

May need to change settings to get 256 colors.

  - `set -g default-terminal "screen-256color"` in `.tmux.conf`
  - `export TERM="screen-256color"` in `.shellrc`
  - PuTTY: Connection > Data > Terminal-type `xterm-256color`
